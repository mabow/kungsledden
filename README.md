# Kungsleden offline GPS map

Open on your phone (HTTPS — GPS works):

**https://mabow.github.io/kungsledden/**

## Install for offline (Firefox)

1. Open the link above and wait until the badge says **offline ready ✓**
2. Firefox menu **⋮ → Install** or **Add to Home screen**
3. Open from the home-screen icon on trail — works with no signal

**Airplane-mode test:** after step 1, turn on airplane mode, close Firefox, reopen the home-screen icon. The map should still load.

Tap **enable GPS** and allow location when prompted.

The map image is embedded in the page; the service worker caches the full app for offline use.
